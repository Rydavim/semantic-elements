Assignment 1 - Written Prompt

Semantic Elements:

Semantic elements clearly convey their meaning to the developer working with the code as well as the browser which interprets and displays the web page. These elements make code more descriptive and can help provide necessary context for accessibility devices. HTML is all about structure and semantic elements provide much more explicit structure compared to blocks and blocks of div tags.
